# 목적

본 문서는 Premium Embedded UX 설계 원칙을 정의한다.

---

# 1. 핵심 철학

좋은 MCU UX는:

```text
화려함
```

이 아니라

```text
신뢰감
```

이다.

---

# 2. 가장 중요한 목표

```text
1초 안에 상태 이해 가능
```

---

# 3. 숫자 우선

현재 온도/습도는:

```text
UI의 중심
```

이어야 한다.

---

# 4. 정보량 제한

```text
❌ 작은 숫자 다량 표시
❌ 엔지니어링 값 남발
```

---

# 5. 좋은 UX 특징

| 특징 | 설명 |
|---|---|
| 차분함 | 안정감 |
| 단순함 | 빠른 이해 |
| 일관성 | 학습 용이 |
| 큰 숫자 | 즉시 인지 |

---

# 6. Ambient UX

권장:

```text
Soft Highlight
Gentle Transition
Tiny Trend
```

---

# 7. 금지

```text
❌ RGB Flash
❌ Bounce
❌ 과한 Motion
```
