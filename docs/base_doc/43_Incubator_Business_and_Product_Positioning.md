# 1. 핵심 철학

```text
DIY 장치가 아니라
완성된 제품처럼 보여야 한다.
```

# 2. 목표 사용자

| 사용자 | 특징 |
|---|---|
| 소규모 농장 | 안정성 중요 |

| 교육 기관 | 시각적 이해 중요 |
| 스마트팜 운영자 | Cloud 중요 |

| 항목 | 방향 |
