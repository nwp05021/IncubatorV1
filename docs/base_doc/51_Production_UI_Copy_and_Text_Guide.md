# 51_Production_UI_Copy_and_Text_Guide

> Version: 1.0
> Status: Strategic

---

# 목적

본 문서는 제품 UI 문구 및 텍스트 가이드를 정의한다.

---

# 1. 핵심 철학

```text
텍스트는 짧고 명확해야 한다.
```

---

# 2. 권장 스타일

| 스타일 | 설명 |
|---|---|
| 짧은 문구 | 빠른 이해 |
| 직접 표현 | 혼란 감소 |
| 상태 중심 | 행동 유도 |

---

# 3. 좋은 예시

```text
SAFE MODE
CHECK SENSOR
WIFI CONNECTED
```

---

# 4. 나쁜 예시

```text
The system encountered an abnormal thermal situation.
```

---

# 5. Alarm 문구

권장:

```text
HIGH TEMP
CHECK HEATER
```

---

# 6. 최종 원칙

```text
사용자는 긴 설명을 읽지 않는다.
```
