# 목적

본 문서는 Premium Product 관점의 최종 리뷰 기준을 정의한다.

---

# 1. 핵심 질문

```text
이 제품이 정말 상용 제품처럼 느껴지는가?
```

---

# 2. UI 리뷰

| 항목 | 확인 |
|---|---|
| Flicker 없는가 | O |
| 큰 숫자인가 | O |
| 정보 과밀 없는가 | O |
| Overlay 명확한가 | O |

---

# 3. UX 리뷰

| 항목 | 확인 |
|---|---|
| 설명서 없이 사용 가능한가 | O |
| 위험 기능 보호되는가 | O |
| 상태 이해 쉬운가 | O |

---

# 4. Recovery 리뷰

| 항목 | 확인 |
|---|---|
| 전원 복구 가능한가 | O |
| SafeMode 동작하는가 | O |
| Alarm 즉시 표시되는가 | O |

---

# 5. Cloud 리뷰

| 항목 | 확인 |
|---|---|
| Offline 동작 가능한가 | O |
| Shadow 구조 명확한가 | O |
